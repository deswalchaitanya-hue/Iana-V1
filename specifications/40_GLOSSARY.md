# 40 — Glossary

## Purpose
Index canonical terms without duplicating their authoritative specifications.

## Terms
- **Action:** planned supported operation; [15_ACTION_ENGINE.md](15_ACTION_ENGINE.md).
- **Attention Inbox:** bounded matters requiring judgment/action; [10_ATTENTION_INBOX.md](10_ATTENTION_INBOX.md).
- **Bounded context:** separately owned domain boundary; [03_ARCHITECTURE.md](03_ARCHITECTURE.md).
- **Capability:** narrow callable operation, not a module; [19_CAPABILITY_SYSTEM.md](19_CAPABILITY_SYSTEM.md).
- **Commitment:** obligation, expectation, deadline, or follow-up; [09_COMMITMENTS.md](09_COMMITMENTS.md).
- **Context snapshot:** ephemeral authorized task context; [13_CONTEXT_ENGINE.md](13_CONTEXT_ENGINE.md).
- **Domain event:** trusted fact inside Core ownership; [26_EVENT_SYSTEM.md](26_EVENT_SYSTEM.md).
- **Engine:** substantial domain coordination; [03_ARCHITECTURE.md](03_ARCHITECTURE.md).
- **HUD / IANA Mode:** functional overlay presentation; [18_SURFACES.md](18_SURFACES.md).
- **Integration event:** sanitized versioned boundary fact; [26_EVENT_SYSTEM.md](26_EVENT_SYSTEM.md).
- **Memory:** retained user-owned knowledge under a contract; [07_MEMORY.md](07_MEMORY.md).
- **Memory contract:** purpose/scope/retention rule for Memory; [07_MEMORY.md](07_MEMORY.md).
- **Module:** trusted ownership and build boundary; [20_MODULE_SYSTEM.md](20_MODULE_SYSTEM.md).
- **Personal Context Graph:** typed temporal relationships among canonical references; [06_CONTEXT_GRAPH.md](06_CONTEXT_GRAPH.md).
- **Plugin:** separately versioned restricted extension; [21_PLUGIN_SYSTEM.md](21_PLUGIN_SYSTEM.md).
- **Project:** durable context for an objective; [08_PROJECTS.md](08_PROJECTS.md).
- **Project State:** rebuildable materialized current-state view; [08_PROJECTS.md](08_PROJECTS.md).
- **Provenance:** source and transformation lineage; [11_PROVENANCE_AND_RECEIPTS.md](11_PROVENANCE_AND_RECEIPTS.md).
- **Provider:** replaceable AI implementation; [23_PROVIDER_SYSTEM.md](23_PROVIDER_SYSTEM.md).
- **Receipt:** immutable evidence of disclosure, flow, or Action outcome; [11_PROVENANCE_AND_RECEIPTS.md](11_PROVENANCE_AND_RECEIPTS.md).
- **Samsung Mode:** conventional One UI-aligned presentation; [18_SURFACES.md](18_SURFACES.md).
- **Surface:** presentation/input channel without domain rules; [17_SURFACE_ENGINE.md](17_SURFACE_ENGINE.md).
